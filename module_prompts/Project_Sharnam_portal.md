# Project Sharnam portal — master brief

PMC construction portal for **Sharnam Project Development Consultants** — Procore-like modules with India PMC workflows (GFC, Final Index, MB/BBS, DPR/WPR).

## Locked UI

Use **ui-2** Graphite / Navy Desk: https://sharnam-portal.onrender.com/ui/2

## Master project setup

On project create: specs, PMC employees, site details, consultants, packages (Civil / PEB / work categories), directory from HRM, vendors. Module enable toggles for office admin. Audit log export per module.

## Integrations

OneDrive / SharePoint (DMS + drawings), Outlook (RFI / meeting mail), MS Project S-curve later. Setup: `docs/M365_SETUP.md`.
