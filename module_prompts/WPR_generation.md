# WPR generation

Weekly pack assembled from live modules (not a separate data silo).

## Typical WPR contents

- Drawing / GFC register snapshot  
- Milestones & planned vs actual  
- Cashflow / budget status  
- Quality NCR + cube summary  
- Safety NCR / observations  
- Hindrance & risk  
- QAP week matrix  

## UI

Reports → **Generate WPR** for week ending date → XLSX/PDF. Pulls from Progress, Cost, Quality, Safety, Drawings.
