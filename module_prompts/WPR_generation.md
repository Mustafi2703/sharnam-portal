# WPR generation

Weekly pack assembled from live modules (not a separate data silo).

## Typical WPR contents

- Drawing / GFC register snapshot + **DrawingCheck** fills  
- Milestones & planned vs actual  
- Cashflow / budget status  
- Quality NCR + cube summary + **QualityInspection** fills + QAP  
- Safety NCR / observations + **Safety** checklist fills  
- Hindrance & risk  
- SiteExecution fills roll into weekly site activity KPIs  

## UI

Reports → **Generate WPR** for week ending date → HTML/PDF. Pulls from Progress, Cost, Quality, Safety, Drawings.
